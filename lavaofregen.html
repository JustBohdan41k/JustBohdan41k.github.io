Lava of regeneration (Minecraft mod)

Minecraft, but lava will give you "Regeneration" effect

Download on Modrinth


Description

Simple data pack that adds a new game mechanic - lava will give you a "regeneration" effect! You can install it for Minecraft 1.20 and up. If someone don't know how to install this data pack, read this:

How to install:

  • Download data pack
  • Launch minecraft
  • Click "Create new world"
  • Click "More" and select "data packs"
  • Drag .zip archive into "data packs" page
  • Enjoy :)

How to install on server:

  • Download data pack
  • Open your server page (Aternos, ScalaCube etc.)
  • Open 'files' page
  • Open world > datapacks folder
  • Drag downloaded data pack into this folder
  • Start the server and enjoy :)
