About

IWBIA (In what biome I am) is a datapack/mod that shows in action bar biome which player is in. It supports all versions and modloaders from 1.20!‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍ ‍ ‍ ‍‍‍ ‍


Please read before downloading


❗ This is the client-side mod, on server will work only datapack!
Mod will only work in singleplayer and on servers where you have permission to use vanilla commands.


Datapack installing


  • In main menu, click "Singleplayer"
  • Create new world
  • In this menu, find button "Data packs"
  • Import this data pack to your version
  • Select data pack in these settings
  • Enjoy :)

Mod installing

  • Download mod
  • Go to .minecraft/mods
  • Drag downloaded file into this folder
  • Instal any modloader
  • Enjoy :)
